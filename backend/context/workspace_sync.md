# Workspace dynamic sync

本机 workspace 是用户与 AI 共同积累思想、项目决定和生活细节的长期知识层。

- 通过 `WORKSPACE_CONTEXT_DIR` 显式配置，不硬编码用户路径。
- 每次相关请求递归读取目录及子目录中的 Markdown，保存后下一轮对话生效。
- workspace 对 AI 只读；可编辑私人记忆仍由独立记忆目录管理。
- 内容不复制进公开仓库，不写入公开默认配置。
- Ollama 本地模型可接收检索到的相关片段。
- 远程模型默认不接收私人片段，除非用户显式授权。
- 不启动无限后台轮询；同步由用户对话触发，行为可预测、可关闭。
